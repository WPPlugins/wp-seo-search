Get cleaner and SEO friendly URLs for your WordPress search like /search/coding+tips instead of /?s=coding+tips

== Description ==
This plugin makes the url for the search results page look like http://site.com/search/keyword

Credit: https://wordpress.org/plugins/nice-search/

== Installation ==
Just upload your plugin in your WP dashboard and activate it. And if you want to change the search base, go to your Permalinks page and enter the one you want.

== Frequently Asked Questions ==
= I want to change the word "search" in the permalink. How? =
Go to Permalinks and change your search base
